# 📦 Coffee Stock (RCS 在庫管理)

모바일 환경과 야외 출점에 최적화된 동아리(서클) 재고 관리 시스템입니다.

## ✨ Features
- **모바일 최적화 UX**: 직관적인 햅틱 스텝퍼(＋/－)와 토스트 알림
- **빠른 접근성**: 즐겨찾기(★) 및 최근 수정한 항목 최상단 자동 배치
- **스마트 검색**: 히라가나 ⇄ 가타카나 자동 호환 검색 및 분류(카테고리) 필터링
- **일괄 처리**: 변동된 여러 항목을 단 한 번의 클릭으로 일괄 저장
- **실시간 알림**: 재고 변경 및 부족 상태를 Discord Webhook으로 즉시 전송
- **데이터 백업**: 현재 재고 및 변경 이력을 CSV 파일로 내보내기

## 🛠 Tech Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Database**: Supabase (PostgreSQL)
- **Deployment**: GitHub Pages
- **Integration**: Discord Webhook API (with CORS Proxy)
